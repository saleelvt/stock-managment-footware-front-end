# Elegance Footwear Stock Manager - Offline Version

## 🎉 Welcome!

This is your portable, offline-ready stock management system for Elegance Footwear. The application works completely offline and doesn't require an internet connection after initial setup.

## 🚀 Quick Start

### For Windows Users:
1. Double-click `start-windows.bat`
2. The application will open automatically in your browser
3. Start managing your inventory!

### For macOS/Linux Users:
1. Open terminal in this folder
2. Run: `./start-unix.sh`
3. Or run: `chmod +x start-unix.sh && ./start-unix.sh`
4. The application will open automatically in your browser

## ✨ Features

- **📦 Complete Inventory Management** - Add, edit, delete products with size-based stock tracking
- **💰 Sales Recording** - Record sales with customer details and automatic stock updates
- **↩️ Return Processing** - Handle returns with stock restoration
- **📊 Dashboard & Reports** - View statistics and low-stock alerts
- **🔍 Search & Filter** - Find products quickly by code, name, or color
- **💾 Offline-First** - All data stored locally, works without internet

## 🛠 System Requirements

- **Windows, macOS, or Linux** operating system
- **Modern web browser** (Chrome, Firefox, Safari, Edge)
- **No internet connection required** after setup!

## 📁 What's Included

- `app/` - The built application files
- `server.js` - Simple server to run the application
- `start-windows.bat` - Windows startup script
- `start-unix.sh` - macOS/Linux startup script
- `README.txt` - This file

## 🔧 Troubleshooting

### Application won't start?
1. Make sure you have a modern web browser installed
2. Check if any other application is using port 3000
3. Try restarting your computer

### Port already in use?
Edit `server.js` and change `const PORT = 3000` to a different number (e.g., 3001)

### Need help?
Contact your system administrator or support team.

## 💡 Tips

- **Bookmark the URL** - Save `http://localhost:3000` in your browser bookmarks
- **Data Persistence** - All your data is stored locally in your browser
- **Multiple Users** - Each computer/user has their own separate data
- **Backup** - Copy the entire folder to backup your data

## 🔒 Security & Privacy
